# DEVChat
DevChat is a collaborative coding platform that bridges developers with AI. It provides a seamless environment for pair programming, code generation, and real-time collaboration, all while maintaining a sleek, retro-futuristic interface.

##  Key Features

- **AI-Assisted Coding**:
  - Generate code snippets, functions, or entire file structures using AI.
  - Ask for code explanations, refactoring suggestions, or optimization ideas.

- **Real-Time Collaboration**:
  - Create private "DevRooms" with unique shareable links.
  - Supports multi-user sessions with synchronized code editing.
  - **Live Chat Interface**: Integrated messaging system to discuss code and ideas.

- **User Experience**:
  - **Cyberpunk Aesthetic**: A stunning "Tron-like" visual design with dark mode, neon accents, and CRT scanlines.
  - **Customizable Profiles**: Users can select from a range of retro-futuristic avatars.
  - **Code Preview**: Instantly visualize generated or edited code in a syntax-highlighted viewer.

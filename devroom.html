Looking at your code on line 30 — the issue is that io.to(roomId).emit will silently fail if the socket hasn't actually joined that room yet. The socket.join(roomId) is async but you're not awaiting confirmation.

Try this pattern instead:
javascript
socket.on('chat_msg', async (data) => {
// broadcast to room first
socket.to(data.roomId).emit('message', data);
if (data.text.startsWith('@ai')) {
const reply = await askAI(rooms[data.roomId], data.text);
io.to(data.roomId).emit('ai_response', { reply, roomId: data.roomId });
}
});
Use socket.to() instead of io.to() for the initial broadcast — it excludes the sender which is usually what you want. Also pass roomId back with the AI response so the client knows which room to update.
